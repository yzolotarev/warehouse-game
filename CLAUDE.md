# warehouse-game

## ИИ-работник склада (протокол «ящик робота»)

В начале рабочей сессии выполни `./bin/wh queue` - это ящик робота на живом
складе Ярослава (сервер на 127.0.0.1:8091, systemd-юнит `warehouse-server`).
Коробки попадают туда разметкой «Робот» (страница /robot_page или разбор
фокуса) и УХОДЯТ из списков человека - ящик полностью мой.

- В ящике есть задачи → сказать юзеру: «у робота N задач» и по его команде
  «приступай» (или явной просьбе поработать) разобрать ящик БЕЗ уточняющих
  вопросов: делаю всё, что могу сделать сам, прямо сейчас.
- Сделал → `./bin/wh done ID "что сделано, где результат"` (закрывает коробку;
  очки не начисляются - экономика меряет энергию человека).
- Не смог/мутно/нужно решение человека → `./bin/wh report ID "стопор: ..."` -
  коробка остаётся в ящике и УТРОМ САМА вернётся в инбокс хозяина с пометкой
  «🤖 было у робота» и этим комментарием. Не спрашивать в чате - стопор в отчёт.
- Правки в этом репо и безобидные скрипты - делать сразу; чужие проекты
  с трейдоффами и внешние сервисы - в стопор с вариантами.
- Побочная мысль/находка → `./bin/wh add "текст"` (уйдёт в инбокс на триаж).
- `./bin/wh focus` - очередь фокуса человека (посмотреть контекст, не трогать).

### Конвейер wh-work (диспетчер ящика)

`./bin/wh-work` - железный скрипт-диспетчер: берёт ящик робота окном по 5,
на КАЖДУЮ коробку запускает отдельную безголовую сессию (`claude -p`) с
приказом из `bin/wh-work-order.md`, проверяет вердикт по базе, пишет
статус (`wh-work status`), отчёт (`wh-work report`) и стенограммы
(`~/.local/share/warehouse/robot_logs/`). Запускает ЧЕЛОВЕК (или Гермес
по его сигналу). В интерактивной сессии конвейер руками не имитировать:
либо протокол ящика выше, либо `wh-work`. `--one` - прогнать одну коробку.

## Код

- Сервер: `server/warehouse_server.py` (FastAPI + sqlite, всё в одном файле),
  страницы - `server/*.html`. Граф кода: `graphify-out/`.
- Новые UI-строки дублировать в словарь PAIRS в `server/i18n.js` (RU/EN).
- После правок сервера: `systemctl --user restart warehouse-server`.
